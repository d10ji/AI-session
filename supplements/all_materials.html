Add Context to a Vague Prompt

Individual exercise → class discussion. Rewrite each vague prompt into a specific, context-rich one.

What to Add
A strong prompt includes: who you are · what you need · the format · a length or constraint
Exercise — Rewrite Each Prompt
#Vague PromptYour Improved Version
1 "Explain databases"
Ideal Answer
"I am a first-year IT student preparing for my database module. Explain what a relational database is in simple terms. Include a one-sentence definition, a real-life example such as a school record system, and two reasons why databases are better than a spreadsheet. Use bullet points. Keep the answer under 150 words."
2 "Help me write an email"
Ideal Answer
"I am an IT student writing to my lecturer to request a one-week extension on my assignment due to unexpected illness. Write a formal but polite email. Include a subject line, a greeting, the reason for the request, the new deadline I am asking for, and a professional closing. Keep it under 150 words."
3 "What is Python?"
Ideal Answer
"I am an IT student who already knows C programming. Explain what Python is in 3–4 sentences. Connect it to C — how is Python similar and how is it different? Mention one real use case where Python is preferred over C. No code examples needed, just a clear explanation in plain language."
4 "Summarize this topic"
Ideal Answer
"I am a first-year IT student studying networking. Summarize the concept of IP addressing. Include what it is, why it matters, and the difference between IPv4 and IPv6. Write it as 4–5 clear bullet points. Use simple language — avoid jargon I would not know as a first-year student."
Example
Vague
"Tell me about networking"
Better
"I am a first-year IT student. Explain computer networking simply. Cover what it is, why it matters, and one real-life example. Use bullet points, under 150 words."
Class discussion: Share your best rewrite. Compare AI's response to the vague vs improved version — how different were the outputs?

5 Steps
1
Ask AI to explain it — 5 min
"Explain Python [concept] to a beginner IT student in 3–4 sentences. Show the simplest possible example."
2
Ask AI for starter code — 5 min
"Give me a very simple Python program using [concept] to solve a small real-life problem. No more than 10 lines."
3
Run it on Replit — 10 min
Create a new Python repl · paste the code · click Run. If there's an error, paste it into AI: "I got this error, what does it mean and how do I fix it?"
4
Make one small change — 5 min
Change a number, a word, or a list item and run again. This proves you understand what the code does.
5
Share with class — 5 min
Which concept? What does your program do in one sentence? What surprised you about using AI to learn?
  • If AI gives complex code, say: "Simplify it — remove anything that isn't needed for a beginner."
  • Errors are normal. Copy the error and ask AI to explain it.
  • It is OK to ask the same question multiple ways until you understand.

Lab 2 — Debug This Program

Exactly 2 bugs. Find them, fix them, and add a comment on each fixed line explaining what you changed.

1
Run the code
Paste into Replit and click Run. Read the error message carefully — the line number tells you where to look.
2
Ask AI about the error
Paste the exact error message into AI: "What does this error mean and where does it come from?"
3
Fix both bugs
Add a comment on each fixed line: what you changed and why. Then run it again to confirm it works.
Broken Code — Student Copy
Python
# This program has 2 bugs. Find them and fix them.

def calculate_average(numbers):
    total = 0
    for num in numbers:
        total = total + num
    average = total / len(number)   # BUG 1 — look at the variable name carefully
    return average

scores = [85, 92, 78, 96, 88]
result = calculate_average(scores)
print("The average score is: " + result)  # BUG 2 — what type is result?
Bug 1: number → numbers — NameError. The parameter is called numbers but the body typed number (missing the s).

Bug 2: result is a float, not a string — TypeError. You cannot join a number and a string with +. Wrap in str(), or use an f-string.
Python — Fixed
def calculate_average(numbers):
    total = 0
    for num in numbers:
        total = total + num
    average = total / len(numbers)   # FIXED: "number" → "numbers"
    return average

scores = [85, 92, 78, 96, 88]
result = calculate_average(scores)
print("The average score is: " + str(result))  # FIXED: str() wraps the float

# Alternative fix for Bug 2 (f-string — also correct):
# print(f"The average score is: {result}")

Reusable Prompt Templates

Three templates that cover almost every situation in this workshop. Keep this open while working.

Template 1 — Learning a New Concept
Use when: learning something for the first time
I am an IT student learning [topic].

Explain [specific concept] to me as if I have never seen it before.
Use simple language and avoid jargon.
Give me:
- A one-sentence definition
- A real-life example I can relate to
- One thing beginners usually get wrong

Keep the total answer under 200 words.
Template 2 — Debugging Code
Use when: your code has an error
I am learning Python and I have an error in my code.

Here is my code:
[paste your code here]

Here is the error message I got:
[paste the exact error message]

Please:
1. Explain what the error means in simple terms
2. Point to the exact line causing the problem
3. Show me the corrected code
4. Explain in one sentence what I did wrong so I don't repeat it
Template 3 — Generating Code
Use when: you need AI to write a program
Write a Python program that [describe what it should do in plain English].

Requirements:
- [requirement 1]
- [requirement 2]
- [requirement 3]

Keep the code:
- Simple and beginner-friendly
- Under [number] lines
- With short comments explaining each section

Do not use any advanced libraries. Only use standard Python.
If the Answer Isn't Good
  • "Simplify this — make it shorter and easier to understand."
  • "Give me a different example."
  • "I don't understand [specific part]. Explain just that part."
  • "Rewrite this without using [word/concept]."
  • "Can you show me a step-by-step version?"

Capstone Lab — Build a Mini Python Program

Solo or pairs. Pick one project. Use AI as your coding assistant. Deliver a working demo.

Choose a Project
Project 1
Grade Calculator
Takes a list of marks and prints the average, highest, and lowest score.
Project 2
Quiz Game
Asks the user 3 multiple-choice questions and shows their score at the end.
Project 3
Name List Manager
Add student names to a list, view the list, and count how many are in it.
Project 4
Number Guessing
Random number 1–10. User guesses until correct. Count how many guesses.
Project 5
File Score Reader
Reads scores.txt, calculates average, prints highest and lowest.
Project 6
Contact Book
Dictionary-based: add contacts and look them up by name.
How to Work
1
Prompt AI
Use Template 3 (Slide 31). Describe your project in plain English. Adjust based on what AI gives you.
2
Run on Replit
Paste the code, click Run, read the output carefully.
3
Fix errors with AI
Copy error messages into AI. Use Template 2. Ask what the error means and how to fix it.
4
Test all features
Try edge cases: empty input, wrong type, large numbers. Make sure nothing crashes.
5
Demo to instructor
Show it running live before the 45 minutes is up.
