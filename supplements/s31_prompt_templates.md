# Reusable Prompt Templates
**AI for IT Students Workshop**

Keep this sheet. These 3 templates work for almost every situation you'll encounter as an IT student.

---

## Template 1: Learning a New Concept

Use when: you need to understand something for the first time, for an exam, or for a project.

```
I am an IT student learning [topic].

Explain [specific concept] to me as if I have never seen it before.
Use simple language and avoid jargon.
Give me:
- A one-sentence definition
- A real-life example I can relate to
- One thing beginners usually get wrong

Keep the total answer under 200 words.
```

**Example using this template:**
> I am an IT student learning networking.
>
> Explain what an IP address is to me as if I have never seen it before.
> Use simple language and avoid jargon.
> Give me:
> - A one-sentence definition
> - A real-life example I can relate to
> - One thing beginners usually get wrong
>
> Keep the total answer under 200 words.

## Template 2: Debugging Code

Use when: your code has an error and you do not know why.

```
I am learning Python and I have an error in my code.

Here is my code:
[paste your code here]

Here is the error message I got:
[paste the exact error message]

Please:
1. Explain what the error means in simple terms
2. Point to the exact line causing the problem
3. Show me the corrected code
4. Explain in one sentence what I did wrong so I don't repeat it
```

**Tips:**
- Always paste the exact error message — do not paraphrase it
- If you have multiple errors, tackle them one at a time
- After AI fixes it, ask: *"Can you explain why this fix works?"*

## Template 3: Generating Code

Use when: you need AI to write a program or a piece of code for you.

```
Write a Python program that [describe what it should do in plain English].

Requirements:
- [requirement 1]
- [requirement 2]
- [requirement 3, etc.]

Keep the code:
- Simple and beginner-friendly
- Under [number] lines
- With short comments explaining each section

Do not use any advanced libraries. Only use standard Python.
```

**Example using this template:**
> Write a Python program that calculates the average of a list of student marks.
>
> Requirements:
> - Accept a list of 5 numbers
> - Calculate and print the average
> - Print which marks are above average
>
> Keep the code:
> - Simple and beginner-friendly
> - Under 20 lines
> - With short comments explaining each section
>
> Do not use any advanced libraries. Only use standard Python.

## Quick Tips for All Templates

| Do | Don't |
|---|---|
| Be specific about what you need | Say "give me something good" |
| Set a length limit | Let AI write 500 lines when you need 10 |
| Ask for one thing at a time | Ask for 5 features in one prompt |
| Paste exact error messages | Try to describe the error in your own words |
| Say your level ("I am a beginner") | Assume AI knows your experience level |
| Ask follow-up questions | Accept the first answer if it's unclear |

---

## If the Answer Is Not Good

Try these follow-up prompts:

- *"Simplify this — make it shorter and easier to understand."*
- *"Give me a different example."*
- *"I don't understand [specific part]. Explain just that part."*
- *"Rewrite this without using [word/concept]."*
- *"Can you show me a step-by-step version?"*
