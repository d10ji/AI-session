# Lab 1: Learn a Python Concept Using AI
**AI for IT Students Workshop — Day 2**
Time: 30 minutes

---

## Your Task

Pick ONE Python concept from the list below. Use AI (ChatGPT, Gemini, or Copilot) to learn it, then write and run a small working program that uses it.

---

## Pick One Concept

| # | Concept | What it does |
|---|---------|--------------|
| 1 | `for` loop | Repeat something multiple times |
| 2 | `if / else` | Make a decision in code |
| 3 | Function (`def`) | Package code so you can reuse it |
| 4 | List | Store many values in one variable |
| 5 | `input()` | Ask the user to type something |

---

## Step-by-Step

### Step 1 — Ask AI to explain it simply (5 min)
Open ChatGPT or Gemini and type this prompt (replace the blank):

> "Explain Python [your concept] to a beginner IT student in 3–4 sentences. Then show me the simplest possible example."

Read the explanation. If anything is unclear, ask a follow-up question.

---

### Step 2 — Ask AI to give you starter code (5 min)
Type this prompt:

> "Give me a very simple Python program that uses [your concept] to solve a small real-life problem. No more than 10 lines."

Copy the code.

---

### Step 3 — Run it on Replit (10 min)
1. Go to **replit.com** and create a new Python repl
2. Paste the code
3. Click **Run**
4. Read the output — does it make sense?

If there is an error, paste the error back into AI and ask: *"I got this error, what does it mean and how do I fix it?"*

---

### Step 4 — Make one small change (5 min)
Change something in the program — a number, a word, a list item — and run it again. This proves you understand what the code does.

---

### Step 5 — Share with the class (5 min)
Be ready to answer:
- Which concept did you pick?
- What does your program do in one sentence?
- What surprised you about using AI to learn?

---

## Deliverable

A short working Python program (running without errors) using the concept you chose.

---

## Tips
- If the AI gives you complex code with things you don't recognize, say: *"Simplify it — remove anything that isn't needed for a beginner."*
- It is OK to ask the same question multiple ways until you understand.
- Errors are normal. Copy the error and ask AI to explain it.
