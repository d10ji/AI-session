# Activity: Write Your First Structured Python Prompt
**AI for IT Students Workshop — Day 1 · Slide 18**
Time: 20 minutes

---

## Instructions for Instructor

Assign one scenario per student or group. Each student writes a structured prompt for their scenario using the Role + Task + Context + Output + Constraints format.

After writing, run 2–3 student prompts live on screen and score them together using the 6-question checklist.

---

## Requirements for Every Prompt

Every prompt must include all five elements:

| Element | What it means |
|---|---|
| **Role** | Tell AI who it is ("Act as a Python tutor...") |
| **Task** | What exactly you want AI to do |
| **Context** | Your background ("I know C programming...") |
| **Output** | Format you want (explain + code example + common mistake) |
| **Constraints** | What AI should NOT do or include |

## Evaluation Checklist (use this to score live)

Score each prompt out of 6 — one point per item:

- [ ] Has a clear Role defined
- [ ] Task is specific — AI knows exactly what to produce
- [ ] Context mentions C background or student level
- [ ] Output format is specified (table, example, explanation, etc.)
- [ ] At least one Constraint is included
- [ ] Language is precise — no vague words like "good" or "nice"
